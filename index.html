Origin Story

Born from BlueWallet.
Forked for BC2.

GenesisWallet is a community-driven open-source fork of BlueWallet, one of the most trusted Bitcoin wallets in the world. We preserved everything that makes BlueWallet excellent — its security model, its clean architecture, its commitment to self-custody — and adapted it for the BC2 network.

BC2 (BitcoinII) is Bitcoin's conservative sister chain, preserving Satoshi's original proof-of-work code with a fresh genesis block. GenesisWallet gives the BC2 community a native, mobile-first wallet built specifically for this chain.

Every element — from the gold-amber color palette to the network parameters — has been carefully adapted. This is not an official wallet — it's a community project, built by the people who use it.

Self-Custody First

Your keys, your coins. GenesisWallet never has access to your private keys or seed phrase. Full sovereignty by default.

Fully Open Source

Every line of code is public and auditable on GitHub. A community fork you can trust, review, and contribute to.

BC2 Native

Purpose-configured for the BitcoinII network — correct chain parameters, Electrum server integration, and BC2 address formats.

Features

Everything you need.
Nothing you don't.

Inheriting BlueWallet's battle-tested feature set, adapted and focused for BC2.

HD Wallet Support

Hierarchical Deterministic wallets with BIP39/BIP44/BIP84 support. Generate unlimited addresses from a single seed.

SegWit Native

Full SegWit (Bech32) support for lower transaction fees and faster confirmations on the BC2 network.

QR Transactions

Send and receive BC2 with QR codes. Set custom amounts, add labels, and share payment requests instantly.

Multi-Wallet

Manage multiple wallets in a single app. Separate personal, mining, and merchant funds with ease.

Electrum Server

Connects to BC2 Electrum servers for lightweight, fast synchronization without running a full node.

Watch-Only Mode

Import xpub keys to monitor wallets without exposing private keys. Perfect for cold storage oversight.
